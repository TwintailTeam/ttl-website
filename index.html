Features

Multi platform

TTL supports both Linux and Windows, providing you seamless experience regardless of your Operating System!

Mod support & QoL

TTL comes bundled with optional support for using mods for certain set of our supported games (enable "XXMI" to use) while also adding Quality of Life features such as FPS Unlocking!

Advanced configs

Control and set your environment variables, prelaunch commands, additional game arguments and more, launcher does not restrict you!

FAQ

We currently support Linux and Windows only. Maybe in the future we add support of macOS.

TTL is free and open source application and will never be paid software. You can check and give it a star on GitHub

We do our best to ensure the launcher avoids detection and doesn't interfere with sensitive files or anti-cheat systems. However, some games require the Launcher to disable the anti-cheat to get them running (Linux only). So use TTL at your own risk.

We currently support all HoYoverse games and all KuroGame games.

If you have a bug to report or want to request a feature head over to our GitHub issues and open an issue with respective tag(s).
If you need support or just want to hangout with community and stay up to date with updates you can join our Discord server
