Features

Everything you need to play

Downloads, updates, mod tooling and launcher tweaks in one place. Less setup, more play.

Truly multi-platform

Native Windows and Linux builds, with packages for winget, Flatpak, AUR, COPR and Terra.

Mod support & QoL

Optional XXMI integration brings mod support to supported titles, plus quality-of-life extras like FPS unlocking, all toggled with a single click.

Advanced configuration

Set environment variables, pre-launch commands, extra game arguments and more. The launcher stays out of the way, and power users keep control.

Smart download manager

Queue installs and updates, watch live network throughput and disk usage, and let efficient binary patching keep your games up to date with minimal downloads.

Private by design

Twintail Launcher collects no telemetry and writes no analytics logs. Your library and play habits stay on your machine.

Free & open source

Built with Tauri, Rust and TypeScript and licensed under GPL-3.0. You can read the code, file issues, or contribute. TTL will always be free.

Download

Get Twintail Launcher

Pick your platform and install method. Always 100% free.

wingetInstall & update via the Windows Package Manager
>winget install TwintailLauncher
Flatpak (Flathub)Works on any distro
$flatpak install flathub app.twintaillauncher.ttl
Arch (AUR)twintaillauncher · -git · -bin
$yay -S twintaillauncher-bin
$paru -S twintaillauncher-bin
Fedora (COPR)Enable the repo, then install
$ sudo dnf copr enable tukandev/TwintailLauncher $ sudo dnf install twintaillauncher
Fedora (Terra)Enable the repo, then install
$ sudo dnf install terra-release $ sudo dnf install twintaillauncher
